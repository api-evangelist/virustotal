More than two years have passed since we announced the launch of Code Insight at RSA 2023 . From that time on, we have been applying this technology in different scenarios , expanding its use in new file formats ( 1 , 2 ). As we advance in the automated analysis of new files with Code Insight, we want to offer an alternative that enables the integration of this type of technology into the analysis of disassembled or decompiled code.
